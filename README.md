# 📊 Marketing Campaign Power BI Dashboard

A comprehensive **Marketing Campaign Analysis Dashboard** built in Power BI to track campaign performance, analyze buyer demographics, and identify key purchase drivers.

---

## 🎯 Project Overview

This dashboard provides actionable insights into marketing campaigns by analyzing:
- **Campaign Performance** – Sales, purchases, and product mix per campaign
- **Buyer Composition** – Demographics, income, and customer behavior
- **Purchase Drivers** – Key factors influencing campaign success and total sales

## 🚀 How to Use This Project

### Option 1: View Dashboard Screenshots
Browse the `Dashboard_Screenshots/` folder for visual previews.

### Option 2: Open in Power BI
1. Download the `.pbix` file from `PowerBI_File/` folder
2. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
3. Open the file and explore interactively

### Option 3: Replicate with Dataset
1. Download dataset from `Dataset/` folder
2. Import into Power BI
3. Build your own version
